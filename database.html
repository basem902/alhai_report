ميزات Supabase

الخدمات السحابية المستخدمة من منصة Supabase

🔐

Auth

مصادقة المستخدمين - تسجيل الدخول بالهاتف، OTP، تسجيل جديد، إدارة الجلسات

⚡

Realtime

المزامنة الفورية - تحديثات مباشرة للطلبات، المخزون، الإشعارات عبر WebSocket

📁

Storage

تخزين الملفات - صور المنتجات، الشعارات، مرفقات الفواتير، النسخ الاحتياطي

⚡

Edge Functions

4 وظائف سحابية - معالجة الطلبات، رفع الصور، إشعارات السائق، webhook التوصيل

🔒

RLS

Row Level Security - سياسات أمان على مستوى الصف لكل جدول حسب المنظمة والدور

🔧

RPC

Remote Procedure Calls - دوال مخزنة للعمليات المعقدة مثل إغلاق الوردية والتقارير

Edge Functions

upload-product-images

notify-driver

تقنيات متقدمة

أبرز الميزات التقنية

تقنيات متقدمة مطبقة في طبقة قاعدة البيانات لضمان الأداء والأمان والموثوقية

🔍

FTS5 - بحث نصي كامل

بحث نصي كامل باستخدام FTS5 للمنتجات - بحث فوري بالاسم، الباركود، والوصف مع دعم اللغة العربية

💾

نسخ احتياطي تلقائي

نسخ احتياطي تلقائي كل ساعتين مع نسخ إضافية قبل وبعد كل عملية ترحيل لضمان استعادة البيانات

🛡

PRAGMA integrity_check

فحص سلامة قاعدة البيانات تلقائيا عند كل فتح للتطبيق للكشف المبكر عن أي تلف في البيانات

🔒

SQLCipher - تشفير كامل

تشفير كامل لقاعدة البيانات المحلية باستخدام SQLCipher لحماية بيانات المبيعات والعملاء على الجهاز

📋

سجل تاريخ الترحيلات

جدول مخصص لتتبع تاريخ جميع عمليات الترحيل مع تفاصيل الإصدار والتوقيت وحالة النجاح

🔄

نسخ ما قبل/بعد الترحيل

إنشاء نسخة احتياطية تلقائيا قبل كل ترحيل وأخرى بعده للتراجع الآمن في حالة حدوث خطأ
